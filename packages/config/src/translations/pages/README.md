# Page Translations

This folder contains modular translation files for individual pages, auto-generated by `scripts/generate-page.js`.

## Structure

Each page has two translation files:
- `{page}.ts` - Slovak translations (SK)
- `{page}.en.ts` - English translations (EN)

## Example

```typescript
// orders.ts (SK)
export const ordersPageSk = {
  pageTitle: 'Objednávky',
  breadcrumb: 'Objednávky',
  columns: {
    id: 'ID',
    customer: 'Zákazník',
  },
  // ...
};

// orders.en.ts (EN)
export const ordersPageEn = {
  pageTitle: 'Orders',
  breadcrumb: 'Orders',
  columns: {
    id: 'ID',
    customer: 'Customer',
  },
  // ...
};
```

## Usage

These modules are auto-imported into main translation files:
- `../sk.ts` imports `ordersPageSk` and adds to `pages.orders`
- `../en.ts` imports `ordersPageEn` and adds to `pages.orders`

## Generation

**DO NOT edit these files manually!**

To regenerate:
```bash
node scripts/generate-page.js scripts/page-configs/{page}-page.json
```

All translation data is defined in the JSON config file.
