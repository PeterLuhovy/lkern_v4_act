# Page Translation Types

This folder contains TypeScript type definitions for page translation modules, auto-generated by `scripts/generate-page.js`.

## Structure

Each page has one type definition file:
- `{page}.types.ts` - TypeScript interface defining translation structure

## Example

```typescript
// orders.types.ts
export interface OrdersPageTranslations {
  pageTitle: string;
  breadcrumb: string;
  columns: {
    id: string;
    customer: string;
  };
  // ...
}
```

## Usage

These types are auto-imported into main `types.ts`:
```typescript
import type { OrdersPageTranslations } from './types/orders.types';

export interface TranslationKeys {
  pages: {
    orders: OrdersPageTranslations;
    // ...
  };
}
```

## Generation

**DO NOT edit these files manually!**

Types are auto-generated from Slovak translation data in JSON config.

To regenerate:
```bash
node scripts/generate-page.js scripts/page-configs/{page}-page.json
```
