## Project Overview

L-KERN v4 is a clean-slate ERP system built with:

- **Frontend**: React 19, TypeScript 5.7, Vite 6
- **Backend**: Python 3.11, FastAPI, gRPC
- **Infrastructure**: Docker, Kafka, PostgreSQL
- **Monorepo**: Nx 20.8
